Eclipse+CDT 配置 Linux 平台 C、C++ 开发环境

发表于 2022-11-06 更新于 2023-01-09 分类于 其他 本文字数： 153 阅读时长 ≈ 1 分钟

Eclipse 开发集成环境常用来开发 Java 项目，如果想通过 eclipse 进行 C、C++ 开发，就需要安装 eclipse 插件 CDT(C/C++ Development Tooling)。

下载 CDT 插件

根据自己的 eclipse 版本，下载对应的 CDT 插件版本：

https://github.com/eclipse-cdt/cdt/blob/main/Downloads.md

安装 CDT 插件

菜单栏 help –> install new software

把解析出来的插件内容全部勾选，然后下一步开始安装，再 accept 版权协议，就完成安装了。
